# Blog web aplication

This is a blog web application where you can register and upload your blogs of any category that you want.

## Initialize the project after clone


Install with npm.

### Client (/) ->
```bash
  1. cd client
  2. npm install 
  3. npm run dev
```

### Server (/)->
```bash
  1. cd server
  2. npm install 
  3. nodemon index.js
``` 
## Deployment

The deployment of this project is completly on Vercel.
